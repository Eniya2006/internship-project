Title:
Startup Resource Allocation System
Problem:
Manual prioritization leads to imbalance and urgency-based decisions.
Solution:
Implemented weighted scoring system considering impact, urgency, customer importance, and effort.
Formula:
Score = (impact × 0.4) + (urgency × 0.3) + (customer_importance × 0.2) − (effort × 0.1)
Features:
Priority calculation
Sorted task list
Resource capacity allocation
Test cases included
